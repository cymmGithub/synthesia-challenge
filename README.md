# Synthesia Webflow Challenge

Custom JS for a Synthesia-branded Webflow page built on the Client-First framework.

## Modules

- **Word Rotator** — animated text cycling in the hero H1
- **Customer Popover** — shared, keyboard-accessible popover on the logo reel
- **Gallery** — masonry grid with shimmer/blur-up loading, powered by external API
- **Modal** — lightbox with focus trap and keyboard navigation

## Setup

```bash
bun install
bun run build   # → dist/synthesia.min.js
```

Production bundle is served via jsDelivr from this repo.
